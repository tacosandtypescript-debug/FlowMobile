Instalador oficial

FlowMobile

Elige tu dispositivo y toca Copiar. Después pega el comando en la aplicación de terminal.

Apple · iPhone/iPad

Para a-Shell completa. Pega el comando y pulsa Return.

curl -fsSL https://raw.githubusercontent.com/tacosandtypescript-debug/FlowMobile/main/bootstrap_ios.py | python3 - tacosandtypescript-debug/FlowMobile

Android

Para Termux. Pega las dos líneas juntas y pulsa Enter.

pkg install -y curl
curl -fsSL https://raw.githubusercontent.com/tacosandtypescript-debug/FlowMobile/main/install.sh | sh -s -- tacosandtypescript-debug/FlowMobile

Cuando termine, abre una ventana nueva y ejecuta flow. Si el navegador no permite copiar, mantén pulsado el comando y elige Copiar.
